# BuyFlow Deploy

BuyFlow ERP 프로젝트의 Kubernetes 배포 설정과 GitHub Actions 배포 workflow를 관리하는 repository입니다.

## Repository 역할

- Kubernetes manifest 관리
- Frontend / Backend Deployment 및 Service 관리
- ConfigMap / Secret 예시 관리
- GitHub Actions를 통한 배포 자동화 관리

## 관련 Repository

- frontend-buyflow
- backend-buyflow

## Directory Structure

```text
buyflow-deploy
 ├─ k8s/
 ├─ scripts/
 ├─ .github/workflows/
 └─ README.md
```
